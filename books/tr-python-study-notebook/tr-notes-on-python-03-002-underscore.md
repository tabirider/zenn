---
title: "Pythonのアンスコまとめ"
---

1.  `x, _, z = func()`
    `tuple`のアンパックで飛ばせる
2.  `func_()`
    どうしても予約語使いたいとき、`class_`。PEP8では`clss`よりマシとしている。
3.  `_val`, `_func()`
    `protected`(のような意思表明)。外から使わないでね(使える)。
4.  `__func()`
    **マングリング発動**、`_ClassName__func()`に内部変換される。
    `private`(のような意思表明)。継承で名前衝突を避ける措置(衝突しないとは言ってない)。
5.  `__func__()`
    **magic method**。再発明するなドキュメント見ろ。
6.  `1_000_000`
    `,`の代わりに使える。
7.  インタプリタの最後の回答を引用
    ```python
    >>> 1
    1
    >>> _ + 1
    2
    ```
